"# Carpeta_Base_De_Datos" 
README — Práctica de Álgebra Relacional y SQL
Descripción
Esta práctica tiene como objetivo aplicar operadores de álgebra relacional para consultar una base de datos, y traducir dichas expresiones a SQL de forma precisa. Se validó la equivalencia entre ambos lenguajes mediante la ejecución de consultas en Relacional y MySQL, asegurando que los resultados coincidan en número y contenido.
Contenido
- Consultas con selección (σ), proyección (π) y unión (⋈)
- Traducción exacta de álgebra relacional a SQL
- Creación de vistas (VIEW) para encapsular lógica compleja
- Validación cruzada entre Relacional y SQL

Conclusión
La práctica fortaleció el razonamiento lógico y técnico para el diseño de consultas, y demostró la importancia de controlar atributos en las uniones para evitar errores. Las vistas generadas permiten reutilizar consultas complejas de forma clara y estructurada.
